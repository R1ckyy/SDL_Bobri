# Bobránci (Bulánci)

To download the game, build it and play it, run:
```sh
git clone https://github.com/R1ckyy/SDL_Bobri.git
cd SDL_Bobri
./build.sh
```
To run the game (after it was built) run:
```sh
./build/sdlgame
```
